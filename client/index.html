How To Play:
WASD & Space controls the wee little spaceship
ArrowKeys play tetris! Enter will reset the tetris field.
Green cubes will heal your ship
Red cubes will recharge the weapons
Blue cubes will recharge the engines
Log on to the same host (ip):(port) to play against each other!
Hit it!
If there are a lot of players in game, you might want to set up
your own server. Grab stuff from: here,
install nodejs, run npm install, and run "node app.js" on the
in the game directory
Navigate to localhost:1337 to play directly on the server, or (ip):1337 for remote.
